Attendance Pro v6.2.3 Professional
----------------------------------

This version includes:
1. GPS Geolocation Tracking for check-in/check-out.
2. Sticky Header/Footer UI for better mobile experience.
3. Enhanced Arabic labels and currency symbols.
4. PWA Support (Manifest and Icons) for better "Online" feel.
5. Live Cloud Sync with Supabase.

How to Deploy:
1. Upload the content of 'dist' folder to Netlify (use the provided zip).
2. Run the SQL script in 'DATABASE_SETUP.sql' in your Supabase SQL Editor.
3. Share the Netlify link with your employees.
4. They can download the APK from the 'About' or 'Settings' page.

For the Admin:
- Login with your admin account.
- Go to 'Attendance' to see employee locations on Google Maps.
- Create employee accounts from the 'Employees' page.

Enjoy the professional tracking!
